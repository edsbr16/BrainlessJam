## Usage
### Purpose of the game
```
Solve problems in each room to open the door to a more challenging room.
```
### Controls
```
# Up arrow
Moves character north

# Down arrow
Moves character south

# Right arrow
Moves character east

# Left arrow
Moves character west
```
